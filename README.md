# NHS Appointment & Social Media Analysis (Python)

## Overview
This project analyses NHS appointment data alongside Twitter (X) data to evaluate service demand, capacity, and patient engagement trends.

## Objectives
- Assess whether NHS capacity meets demand
- Evaluate efficiency of resource utilisation
- Explore public sentiment using social media data

## Tools & Techniques
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Data cleaning, transformation, and aggregation
- Time-series and categorical analysis

## Key Insights
- High and seasonal demand, especially in autumn/winter
- Capacity operates close to limits during peak periods
- Strong dependency on GP services
- Missed appointments reduce efficiency
- Digital healthcare methods remain underutilised

## Recommendations
- Increase workforce flexibility during peak demand
- Expand use of non-GP healthcare professionals
- Reduce missed appointments through reminders
- Promote digital consultations
- Improve distribution across service settings
